# Dotfiles

This repository contains configuration files, scripts, and utilities for various tools

## What's Included

- **Neovim**: a highly customized Neovim setup for development
- **Alacritty**: configuration for the Alacritty terminal emulator
- **Starship**: configuration for the Starship cross-shell prompt

## Prerequisites

Before setting up, make sure you have the following tools installed:

- [Neovim](https://neovim.io/) (v0.9.0 or later recommended)
- [Alacritty](https://alacritty.org/)
- [Starship](https://starship.rs/)
- [GNU Make](https://www.gnu.org/software/make/)
- A [Nerd Font](https://www.nerdfonts.com/) for proper icon rendering in the terminal and editor
